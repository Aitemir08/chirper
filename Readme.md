Chirper
======

Chiper is a more intuitive version of Twitter

## Setup

1. Clone thr repository

```
git clone http://github.com/Aitemir08/chirper
```

2. Set configuration parameters such as

```
CHIRPER_DB_HOST=
CHIRPER_DB_PORT=
CHIRPER_DB_NAME=
CHIRPER_DB_USER=
CHIRPER_DB_PASS=

CHIRPER_DB_PORT=
```

in shell or in `.env` file.

3. Install libraries

```bash
npm istall
```
4. Start server.

```bash 
node chirper.js
```
